snake
=====

A simple ncurses snake game in C.

Compiling
---------

        $ make

Running
-------
Make sure the highscore file (snakehighscore) is in the current directory, 
then run

        $ ./snake
        
Usage
-----
Use the arrow keys to control the snake. Eat the food, don't run into yourself or the walls.
 - p Pause
 - q Quit
 - t Highscore table
