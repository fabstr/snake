snake
=====

A simple ncurses snake game in C.

Compiling
---------

    $ make

Running
-------
Make sure the highscore file exists at $HOME/.snakehighscore with the contents

    Count=0

then run

    $ ./snake

Usage
-----
Use the arrow keys to control the snake. Eat the food, don't run into yourself or the walls.

Keys:
 - h help
 - p Pause
 - q Quit
 - t Highscore table
